Golden Sands is a property developer and holiday rental company located in sunny south Goa. They wanted a website which would give them a web presence and give their customers an easy way to contact them. The website is very photo heavy. Each photo links to a full page gallery which is indicated by a hover zoom. Each property has its own page with each available room listed. The website has been designed to function similarly to very popular online property booking websites for user familiarity and comfort.
